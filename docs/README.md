# Taia Tiniyara - Portfolio & Blog

This is the personal portfolio and blogging website for Taia Tiniyara, a software engineer.

## About

This website serves as:
- **Portfolio**: Showcasing my software engineering projects and professional work
- **Blog**: Sharing insights, tutorials, and thoughts on software development

## Tech Stack

- React with TypeScript
- Vite
- TanStack Router
- Supabase
- Tailwind CSS
- Shadcn/ui components

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy
npm run deploy
```
